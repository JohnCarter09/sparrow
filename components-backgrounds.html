Parallax Backgrounds

Use .parallax class with .background-holder class and .has-parallax to it's parent for effect.

JavaScript
<script src="assets/lib/rellax/rellax.min.js"></script>
Image Background Parallax
<section class="color-white text-center has-parallax">
	<div class="background-holder overlay overlay-2 parallax h-100vh" style="background-image:url(assets/images/header-03.jpg);" data-rellax-speed="-10">
	</div>
	<div class="text-center">
		<h5>Image Background Parallax</h5>
	</div>
</section>
Video Background Parallax
<section class="color-white text-center has-parallax" id="header-video">
	<div class="background-holder overlay overlay-2 parallax h-100vh" style="background-image:url(assets/images/headers/header-coming-soon.png);" data-rellax-speed="-10">
		<video autoplay="autoplay" loop="loop" muted="muted">
			<source src="assets/videos/demo/video.mp4" type="video/mp4"/>
		</video>
	</div>
	<div>
		<h5>Video Background Parallax</h5>
	</div>
</section>
